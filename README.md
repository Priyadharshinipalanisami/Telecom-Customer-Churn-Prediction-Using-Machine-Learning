## 📌 Project Overview

Customer churn is one of the biggest challenges for subscription-based businesses. This project uses **Machine Learning** to predict whether a customer is likely to leave a service based on customer demographics, account details, and service usage.

The project follows the complete machine learning workflow—from data preprocessing to model evaluation—to help businesses identify at-risk customers and improve customer retention strategies.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/(https://github.com/Priyadharshinipalanisami)/customer-churn-prediction.git
```

Move into the project directory

```bash
cd customer-churn-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the notebook

```bash
jupyter notebook customer.ipynb
```
